# Ecommerce API - Auth & Token Testing

## Tools Used
- Postman
- DummyJSON API (https://dummyjson.com)

## What This Project Covers
  - Login API (POST) with credentials
  - Token extraction from response using scripts
  - Bearer Token Authorization for secured endpoints
  - GET User Profile with Auth header
  - GET Products API with dynamic data extraction

## How to Run
1. Import the .json collection into Postman
2. Set base_url environment variable to https://dummyjson.com
3. Run Login request first to get token
4. Run remaining requests
